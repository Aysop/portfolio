Accessibility

This portfolio was created in accordance to Web Accessibility Inititive's guidlines and aims to accomodate a range of potential impairments. Formatting accomodations included are: captions for multimedia content, text alternatives for non-text content, and large lettering. In addition, the website is free of contrast errors, which has been verified by the Web Accessibility Evaluation Tool (WAVE), and is fully navigable using the browsers built-in keyboard functions. One simply needs to use tab to jump between interactable content, enter to select, and arrow keys to scroll the entirity of the page and its text content.
